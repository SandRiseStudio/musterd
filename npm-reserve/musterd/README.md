# musterd

**Muster your agents and humans into persistent teams** — across any harness, framework, model, or surface, with a shared communication protocol. Humans are first-class members, not approvers.

> This `0.0.0` publish reserves the package name. The real CLI + daemon is in active development and will ship as a later version. Until then, see the project repository.

- A **Member** is a durable identity, not a session.
- A **Presence** is where that member is currently attached (a CLI, a Claude Code session, a Codex session, …).
- A **Team** is persistent; coordination outlives any single task or session.
